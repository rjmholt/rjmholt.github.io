Current Projects

Toy Compiler

A compiler for a toy language (called Bean) implemented in OCaml, targeting the Oz virtual machine from the University of Melbourne. Eventually, I'm hoping to build on this project to create a more serious compiler in Haskell, targeting a real world architecture like ARM, MIPS or x86_64.

Chat Server

A chat server with rooms with a JSON protocol, implemented in Erlang, following OTC principles. I'm aiming to extend this over time to add TLS support, authentication, peristence and replacing rooms with subscribed groups.

MIPS emulator

A simple toy disassembler, and then emulator, for the MIPS architecture, which takes raw MIPS machine code and can both construct a disassembled representation, and emulate its execution.
